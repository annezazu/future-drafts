# Future Drafts

Drafts for your future self.

A WordPress plugin that adds a dashboard widget for capturing time-delayed drafts. Drop a quick title and a few notes at the moment of an experience, pick a date, and the draft will resurface in the same widget when that date arrives — ready for you to finish writing about it.

See [`docs/plans/2026-04-25-future-drafts-design.md`](docs/plans/2026-04-25-future-drafts-design.md) for the full design.

## Status

Pre-implementation. Design approved, scaffold in place.

## Companion plugin

Works alongside [Draft Sweeper](https://github.com/annezazu/draft-sweeper/) — a related issue will exclude Future Drafts from Draft Sweeper's resurfacing logic.
